<h1 style="color:#1E90FF; font-size:44px; font-weight:900; text-align:center; text-shadow:3px 3px #333; letter-spacing:1px;">DeSocialX</h1>

<p><strong>DeSocialX</strong> is a decentralized social media platform that integrates AI-powered learning, blockchain governance, and gamified engagement. Users control their data and turn social media activity into productive learning with rewards.</p>

<hr>

<h2 style="color:#FF4500; font-size:34px; font-weight:900; text-shadow:2px 2px #555;">Problem</h2>
<p>Traditional social media promotes passive engagement, provides no data control, lacks meaningful rewards, and centralizes governance, leaving users without productive outcomes.</p>


<h2 style="color:#32CD32; font-size:34px; font-weight:900; text-shadow:2px 2px #555;">Our Solution</h2>
<p>DeSocialX enables users to:</p>
<ul>
<li>Manage profiles and data securely on blockchain</li>
<li>Access AI-powered learning modules</li>
<li>Earn NFT badges and maintain streaks</li>
<li>Participate in DAO governance</li>
<li>Engage with a reward-based social feed</li>

<h2 style="color:#1E90FF; font-size:34px; font-weight:900; text-shadow:2px 2px #555;">Features</h2>
<ul>
<li>Decentralized social feed with multimedia</li>
<li>Blockchain-based identity and profiles</li>
<li>AI-powered learning modules with quizzes</li>
<li>NFT badges and streak tracking</li>
<li>DAO for community governance</li>
<li>Users pay fees for blockchain transactions</li>
</ul>

<h2 style="color:#FF6347; font-size:34px; font-weight:900; text-shadow:2px 2px #555;">Getting Started</h2>


<h3 style="color:#1E90FF; font-size:28px; font-weight:900; text-shadow:1px 1px #444;">Prerequisites</h3>
<ul>
<li>Node.js and npm</li>
<li>Python 3.12+</li>
<li>MongoDB</li>
<li>MetaMask wallet</li>
</ul>


<h3 style="color:#32CD32; font-size:28px; font-weight:900; text-shadow:1px 1px #444;">Frontend Setup</h3>
<pre><code>cd frontend
npm install
npm run start
</code></pre>
<p>The app runs at <a href="http://localhost:3000">http://localhost:3000</a></p>


<h3 style="color:#FFD700; font-size:28px; font-weight:900; text-shadow:1px 1px #444;">Backend Setup</h3>
<pre><code>cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
</code></pre>
<p>FastAPI backend runs at <a href="http://127.0.0.1:8000">http://127.0.0.1:8000</a></p>
